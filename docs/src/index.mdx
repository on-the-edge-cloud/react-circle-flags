# react-circle-flags

## Install

```bash
npm install --save react-circle-flags
```

## Basic usage

<Playground>
  <CircleFlag countryCode="es"/>
</Playground>

You can pass all the react's `React.DetailedHTMLProps<React.ImgHTMLAttributes<HTMLImageElement>, HTMLImageElement>` props to CircleFlag.
